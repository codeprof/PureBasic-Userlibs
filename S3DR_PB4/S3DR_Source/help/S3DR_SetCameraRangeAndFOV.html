S3DR_SetCameraRangeAndFOV()

Syntax

S3DR_SetCameraRangeAndFOV(Near.f,Far.f,FOV.f)

Description

This command changes the near- and the far plane and the FOV(Field of View).
The parameter FOV.f must be in degree.

This command must be inside of a S3DR_BeginReal3D()...S3DR_EndReal3D() block.

Supported OS

Windows
