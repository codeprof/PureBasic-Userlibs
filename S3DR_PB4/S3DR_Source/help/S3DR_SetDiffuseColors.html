S3DR_SetDiffuseColors()

Syntax

S3DR_SetDiffuseColors(RGB1.l,RGB2.l,RGB3.l,RGB4.l)

Description

This command sets the diffuse-color values for S3DR_Draw3D() and S3DR_DrawBillboard().
The standard is S3DR_SetDiffuseColors(#white,#white,#white,#white).

This command must be inside of a S3DR_BeginReal3D()...S3DR_EndReal3D() block.

Supported OS

Windows
