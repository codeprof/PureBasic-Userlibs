S3DR_TestCollison()

Syntax

S3DR_TestCollision(O1X1.f,O1Y1.f,O1Z1.f,O1X2.f,O1Y2.f,O1Z2.f,O1X3.f,O1Y3.f,O1Z3.f,O1X4.f,O1Y4,O1Z4.f,O2X1.f,O2Y1.f,O2Z1.f,O2X2.f,O2Y2.f,O2Z2.f,O2X3.f,O2Y3.f,O2Z3.f,O2X4.f,O2Y4,O2Z4.f)

Description

This command checks for box-collision between 2 quads.
If there was a intersection the result is nonzero.

This command must be inside of a S3DR_BeginReal3D()...S3DR_EndReal3D() block.

Supported OS

Windows
